# Tesis.com.co - Sitio Web Estático

Sitio web completamente en HTML, CSS y JavaScript para asesorías en trabajos de grado.

## Estructura del Proyecto

```
tesis-html/
├── index.html                 # Página de inicio
├── css/
│   └── styles.css            # Estilos CSS
├── js/
│   └── main.js               # Funcionalidad JavaScript
└── pages/
    ├── acerca-de.html        # Página Acerca de
    ├── filosofia.html        # Página Filosofía
    ├── politicas.html        # Página Políticas y Condiciones
    ├── servicios.html        # Página Servicios (listado)
    ├── contacto.html         # Página Contacto con formulario
    ├── blog.html             # Página Blog
    └── servicios/
        ├── investigacion.html
        ├── normas-y-estilos.html
        ├── redaccion-y-estilo.html
        ├── defensa.html
        ├── ciencias-sociales.html
        ├── mercadeo.html
        ├── estadistica.html
        └── analisis-datos.html
```

## Características

- ✅ Sitio 100% estático (HTML, CSS, JavaScript vanilla)
- ✅ Navegación responsiva con menús desplegables
- ✅ Formulario de contacto funcional
- ✅ Diseño profesional con colores azul y verde
- ✅ Tipografía moderna (Poppins y Open Sans)
- ✅ Footer con información de licencia Creative Commons
- ✅ Todas las páginas solicitadas implementadas
- ✅ Enlaces internos correctamente configurados

## Páginas Incluidas

### Principales
- **Inicio (index.html)**: Página de bienvenida con hero section
- **Acerca de**: Información sobre la empresa
- **Filosofía**: Principios y valores
- **Políticas y Condiciones**: Términos de servicio
- **Servicios**: Listado de todos los servicios
- **Contacto**: Formulario de contacto
- **Blog**: Artículos y recursos

### Servicios Especializados
1. Investigación
2. Normas y Estilos
3. Redacción y Estilo
4. Defensa de Tesis
5. Ciencias Sociales
6. Mercadeo
7. Estadística
8. Análisis de Datos

## Cómo Usar

1. Abre `index.html` en tu navegador web
2. Navega usando el menú principal
3. Todos los enlaces están configurados correctamente
4. El formulario de contacto es funcional (muestra confirmación al enviar)

## Personalización

Para modificar contenido:
- Edita los archivos `.html` directamente
- Cambia colores en `css/styles.css` (variables: #4A7BA7 azul, #5BA85C verde)
- Modifica funcionalidad en `js/main.js`

## Notas Técnicas

- Responsive design: funciona en desktop, tablet y móvil
- Navegación sticky (se mantiene visible al desplazarse)
- Menús desplegables con CSS puro
- Formulario con validación básica
- Todos los enlaces internos usan rutas relativas
